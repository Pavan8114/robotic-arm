Dual-Mode Robotic Arm

Precision meets automation for the future of robotics.

About the Project

This project develops a dual-mode robotic arm that can be controlled manually with precision or operated automatically using deep learning for object detection. The system enhances flexibility and efficiency, making it valuable for various industries like manufacturing and healthcare.

Key Features

Mimic Control

In this mode, the user manually directs the robotic arm using a smaller, hand-operated arm equipped with potentiometers. The potentiometers capture position and motion data.

Automated Mode

The system captures images using a camera, processes them with a Raspberry Pi, and employs deep learning algorithms to detect objects and calculate their coordinates. The robotic arm then adjusts its servo motors to interact with the identified objects based on the calculated angles.

Project Outcomes

Methodology

Our methodology combines mimic control and automated object detection to ensure accurate and efficient robotic operation.

Result

The system demonstrated seamless integration between manual and autonomous modes, significantly enhancing flexibility and precision.

Summary

This dual-mode robotic arm is a groundbreaking tool designed to excel in various industrial and research applications.

Conclusion

In conclusion, this robotic arm project exemplifies the integration of cutting-edge technologies to achieve exceptional precision and functionality. By employing Arduino and Raspberry Pi microcontrollers, the arm operates in two distinct modes: a mimic mode for real-time manual control and an automatic mode for executing tasks autonomously. The system utilizes NRF24L01 RF modules to ensure reliable wireless communication, which is essential for effective remote control and coordination. Power management is efficiently handled through a combination of DC-DC converters and power supply adapters, ensuring stable power delivery to all components. The PCA9685 servo motor driver, along with MG996R metal gear servo motors, enables precise manipulation of the arm's movements.
